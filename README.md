# 266.-Palindrome-Permutation

Description
Given a string, determine if a permutation of the string could form a palindrome.

Example
Example1

Input: s = "code"
Output: False
Explanation: 
No solution
Example2

Input: s = "aab"
Output: True
Explanation: 
"aab" --> "aba"
Example3

Input: s = "carerac"
Output: True
Explanation: 
"carerac" --> "carerac"
